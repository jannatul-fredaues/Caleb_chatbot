# Candy-AI
This project is a simple chatbot designed to interact with users and provide automated responses based on predefined rules or AI models. It aims to enhance user experience by offering quick, accurate, and interactive communication. 

Features:
1. Real-time user interaction
2. Easy-to-use interface
3. Customizable responses
4. Scalable and lightweight

Technologies Used:
1. Python
2. NLP / AI techniques
3. Basic HTML+CSS

Purpose:
The chatbot can be used for learning, customer support, or as a foundation for building advanced conversational AI systems.
